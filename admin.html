How to publish price changes
  1. 1. Edit any prices below, then click ↓ Download prices.json in the top bar.
  2. 2. Go to your GitHub repo → find prices.json → click the pencil ✏️ edit icon.
  3. 3. Select all the text, paste the downloaded file content, then click Commit changes.
  4. 4. The live site updates in ~60 seconds automatically.
